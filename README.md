#  WAREHOUSE MANAGEMENT (QUẢN LÝ KHO)

> **Dự án:** Hệ thống quản lý kho hàng
> **Tác giả:** Vũ Chiến Thắng

##  Giới thiệu
Ứng dụng quản lý quy trình nhập - xuất - tồn kho, được xây dựng tối ưu với Spring Boot.

##  Công nghệ sử dụng
- **Backend:** Java Spring Boot, Spring MVC
- **Database:** Spring Data JPA, SQL Server (hoặc MySQL)
- **Frontend:** Thymeleaf / HTML5

## Tính năng chính
- Quản lý thông tin thiết bị/sản phẩm.
- Xử lý nghiệp vụ Nhập kho & Xuất kho.
- Báo cáo, thống kê tồn kho theo thời gian thực.
